## Notice
* 用户名必须唯一
* login样式不支持360浏览器兼容模式

* button事件会刷新页面
* $.parseJSON()函数用于将格式完好的JSON字符串转为与之对应的JavaScript对象。
* 在返回的json数据中不要使用status
* bootstrap的Glyphicons字体图标需要在resource下建立fonts
